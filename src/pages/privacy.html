PoE2Dire Privacy Policy

Effective date: May 26, 2026

Summary

PoE2Dire does not collect, sell, share, or transmit personal information.

What the extension reads

PoE2Dire runs only on Path of Exile forum pages when activated by the user. It reads the visible patch-note content on the current page so it can render that content in a different layout.

Local storage

PoE2Dire uses browser storage only to cache resolved wiki icon URLs and missing-icon results. This cache is stored locally in the browser and is used to reduce repeated requests to the wiki APIs. It does not contain personal data, browsing history, account data, or user-generated content.

Network requests

PoE2Dire may request data and images from poewiki.net and poe2wiki.net to display item, skill, passive, monster, and ascendancy icons in rendered patch notes. These requests are only for extension functionality. PoE2Dire does not intentionally send personal data, browsing history, account information, or user-generated content to these services.

Remote code

PoE2Dire does not load or execute remote JavaScript or WebAssembly. All extension JavaScript is included in the extension package. Fetched wiki data and image assets are not executed as code.

Analytics and advertising

PoE2Dire does not use analytics, advertising, tracking pixels, or behavioral profiling.

Data sharing

PoE2Dire does not sell, rent, or share user data.

Data removal

You can remove PoE2Dire's local cache by uninstalling the extension or clearing the extension's storage in your browser settings.

Limited Use

The use of information received from Chrome APIs complies with the Chrome Web Store User Data Policy, including the Limited Use requirements.

Contact

For privacy questions, open an issue at github.com/aisatan/PoE2Dire/issues.
